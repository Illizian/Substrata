Hello world! This is Substrata (or Substrap).

A lightweight, minimalist, responsive, semantic grid powered, unstyled, front-end boilerplate.

What is it?

An attempt to create a bare-bones frontend foundation upon which new projects can quickly and easily be built.

What's included?

It's inspired heavily by and borrows code from Twitter Bootstrap, HTML5 Boilerplate and Semantic Grid System amongst others. Also includes jQuery.

What's not?

Any pre-defined UI or designed components, beyond very basic default typography, spacing and action buttons.

Why are you doing this?

There are some superb tools and frameworks around that help us work faster and smarter. Putting each piece in place to begin a new project can take some time and so far I haven't seen a boilerplate that sets up a blank canvas with everything ready to go.

What's the problem?

Bootstrap is heavy, and contains lots of ready-made components that have to be redesigned. It's often overkill; too much. HTML5 Boilerplate comes with some great page defaults but isn't enough to get you working on what makes this project unique right away. You have to create directories, copy standard files, set up fonts, grab common code snippets, etc.

I want something that remains lightweight but that is powerful: a grid system that leaves my markup alone, a useful set of essential mixins, a stark template that serves as a blank canvas, the extras I need to support older browsers, and ready to go build tools. I want to clone it, run a few setup commands, and begin work.

This is Substrata

A barebones skeleton for your next web project. A set of Less, JS and HTML starting files that target the sweet spot between too much and too little. A set of configuration files to get you up and running quickly with a reasonably modern workflow. It gets you going, and gets out of the way.

Key Features

  • Very lightweight (about 8Kb minified)
  • Simple, Responsive and Semantic Grid
  • No design styles...
  • ...just basic clean typography
  • Google Web Fonts Code
  • Analytics Code
  • Example Media Query Break Points (that can easily be changed)
  • HTML5 Modernizr.js to support older browsers
  • HTML5 Boilerplate .htaccess file
